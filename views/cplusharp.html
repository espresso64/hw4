C++ and C# are two object oriented languages in the C family of
of programming languages. While similar in name and in some
regard, fuctionality, the application of these languages in
game development is wildly varied.

C++ has been regarded as one of the most common languages in game
development. It has been used in games such as Starcraft,
Counter-Strike, and Maplestory. Due to its ability for low level
functionality, the responsiveness and speed bestowed by the
language has been beneficial in creating game engines. However,
this benefit is not limited to PC games. Consoles such as
Playstation and Xbox depend on C++ game code.

C# on the other hand has found a lot of popularity as a scripting
language in popular game engines such as Unity. This language
has a programmer friendly reputation, and gives developers the
flexibility of rapidly developing games.
